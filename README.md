# 템 캘린더

이 폴더에는 GitHub Pages에 바로 올릴 수 있는 웹앱이 들어 있다.

## 들어간 기능
- 관리템 추가, 수정, 삭제
- 오늘 쓸 템 확인 및 체크/해제
- 달 이동 버튼
- 4주 캘린더
- record 화면
- 사진 추가
- 카메라로 바로 촬영
- 날짜별 사진 기록

## GitHub Pages 올리는 방법
1. 이 저장소를 GitHub에 올린다
2. GitHub 저장소 설정에서 `Pages`를 연다
3. 배포 소스를 `main branch`의 `root`로 선택한다
4. 저장 후 잠깐 기다린다
5. 뜨는 주소를 아이폰 Safari에서 연다
6. 공유 버튼에서 `홈 화면에 추가`를 누른다

## 참고
- 이 앱은 `HTTPS`에서 가장 잘 동작한다.
- `file://`로 직접 여는 방식보다 GitHub Pages가 더 안정적이다.
- 아이폰에서 앱처럼 쓰려면 홈 화면 추가가 가장 편하다.
